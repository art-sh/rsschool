Vue.js

Description

Vue.js - one of the most popular javascript frameworks, used to create web interfaces using the MVVM architecture template (Model-View-View-Model)

Functionality

  • Reactive interfaces
  • Declarative rendering
  • Data binding
  • Directives
  • Template Logic
  • Components
  • Event Handling
  • Properties
  • CSS transitions and animations
  • Filters

Installation

              
                
                <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
              

              
                // via npm
                $ npm i vue            // installing module

                import Vue from 'vue'; // import module

                const app = new Vue({
                  el: '#id',          // root element selector
                  data: {
                    name: 'Vue',      // component data object
                    ...
                  },
                  methods: {          // component methods object
                    getName() {
                      return this.name;
                    },
                    ...
                  },
                });

Property binding

              
                

                  
                


                <script>
                  const app = new Vue({
                    el: '#app',
                    data: {
                      blockStyle: '',
                    },
                    methods: {
                      setGreenText() {
                        this.blockStyle = 'color: green';
                      },
                    },
                  });
                </script>
